   🔒 SECURITY IMPLEMENTATION
═══════════════════════════════════════════════════════════════════

✅ 5 SECURITY LAYERS IMPLEMENTED:

LAYER 1: Signature Verification
  - HMAC SHA512 với hash_equals()
  - Tự động reject invalid signature

LAYER 2: Idempotent Processing
  - Check cache/database processed
  - Prevent duplicate processing

LAYER 3: Amount Validation
  - Compare callback amount với database
  - Reject nếu không khớp

LAYER 4: Database Locking
  - SELECT FOR UPDATE trong transactions
  - Prevent race conditions

LAYER 5: Transaction Deduplication
  - firstOrCreate với unique constraints
  - Prevent duplicate records
