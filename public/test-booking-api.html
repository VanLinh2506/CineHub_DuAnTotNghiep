3. Database Info

Kiểm tra dữ liệu trong database bằng các query sau:

Query 1: Kiểm tra Movies

SELECT id, title, status FROM movies WHERE status = 'Chiếu rạp' LIMIT 5;

Query 2: Kiểm tra Theaters

SELECT id, name FROM theaters LIMIT 5;

Query 3: Kiểm tra Showtimes

SELECT s.id, s.movie_id, s.theater_id, s.screen_id, s.show_date, s.show_time, sc.screen_name
FROM showtimes s
LEFT JOIN theater_screens sc ON s.screen_id = sc.id
WHERE s.show_date >= CURDATE()
LIMIT 10;
